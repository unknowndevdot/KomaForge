# KomaForge

**버전 0.1.0**

Windows용 C# WPF **만화/코믹 페이지 에디터**입니다. 페이지에 칸(panel)을 배치하고, 각 칸에 이미지와 말풍선을 넣어 만화 페이지를 구성한 뒤 이미지로 내보낼 수 있습니다.

## 주요 기능

### 페이지·칸
- `1,2,1` 같은 줄별 칸수 입력으로 페이지 자동 생성 (여백·간격 조절)
- 여러 페이지 관리 및 페이지 이동(PgUp/PgDn)
- 칸 드래그 이동, 우하단 핸들로 크기 조절, 빈 영역·테두리 드래그로 칸 이동
- 칸 사변형(quadrilateral) 모서리 모드 — 칸을 비스듬한 사각형으로 변형
- 페이지/칸 단위 배경색, 칸 잠금

### 이미지
- 드래그&드롭 또는 불러오기로 칸에 이미지 배치
- 칸 경계(사변형 포함)에 맞춰 자르기(crop) ON/OFF
- 마우스 휠로 확대/축소, 드래그로 위치 이동, 잠금
- 애니메이션 GIF/WebP, 동영상 프레임 지원
- 실행 파일/저장 파일 폴더 하위의 이미지는 **상대 경로로 저장** → 폴더째 옮겨도 그대로 열림

### 말풍선
- 한 칸에 여러 말풍선, 드래그 이동·크기 조절
- 모양: 원형/사각, 구름/폭발, 파도/외침, 플래시, 집중선, 속도선, 테두리 없음
- 꼬리(tail) — 세 점(시작·중간·끝)으로 곡선 조절, 안으로 깎기, 여러 개 추가
- 겹친 말풍선의 외곽선 자동 병합
- 글자색·배경색·아웃라인색, 글자 아웃라인, 글자 크기 조절

### 저장·내보내기
- 프로젝트 저장/불러오기(`.kfjson`), **Ctrl+S 덮어쓰기 저장**
- 작업 자동 저장 및 다음 실행 시 복원
- 설정·자동저장을 실행 파일과 같은 폴더에 보관(포터블)
- 페이지를 이미지 파일로 내보내기

## 단축키

| 키 | 동작 |
| --- | --- |
| `Ctrl+Z` / `Ctrl+Y` | 실행취소 / 다시실행 |
| `Ctrl+S` | 현재 파일에 덮어쓰기 저장 |
| `Ctrl+R` | 선택 대상 리셋 |
| `L` | 선택 오브젝트 잠금/해제 |
| `Esc` | 선택 해제 |
| `Delete` | 선택한 이미지·말풍선 삭제 |
| `PgUp` / `PgDn` | 이전 / 다음 페이지 |
| `↑` / `↓` | 선택 대상 순서 이동 |
| `마우스 휠` | 크기 조절 |
| `Ctrl+클릭` | 잠금된 오브젝트도 선택 |

## 실행 준비

이 프로젝트는 .NET 8 WPF 앱입니다.

1. Windows용 .NET 8 SDK를 설치합니다.
2. 프로젝트 폴더에서 아래 명령을 실행합니다.

```powershell
dotnet run
```

## 단일 실행 파일 빌드

자체 포함 단일 실행 파일(설치 불필요)을 만들려면:

```powershell
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true
```

생성 위치: `bin/Release/net8.0-windows/win-x64/publish/KomaForge.exe`
