# KomaForge

**버전 0.1.0**

**망가 페이지 에디터**입니다. 페이지에 칸을 배치하고, 각 칸에 이미지와 말풍선을 넣어 페이지를 구성한 뒤 이미지로 내보낼 수 있습니다.

## 주요 기능

### 페이지·칸
- 여백·간격 조절(스냅 기능)
- 여러 페이지 관리 및 페이지 이동(PgUp/PgDn)
- 칸 드래그 이동, 우하단 핸들로 크기 조절, 빈 영역·테두리 드래그로 칸 이동
- 칸 사변형 모서리 모드(칸을 비스듬한 사각형으로 변형)

### 이미지
- 드래그&드롭 또는 불러오기로 칸에 이미지 배치
- 칸 경계(사변형 포함)에 맞춰 자르기
- 마우스 휠로 확대/축소, 드래그로 위치 이동
- 애니메이션 GIF/WebP, 동영상 프레임 지원
- 실행 파일/저장 파일 폴더 하위의 이미지는 **상대 경로로 저장**(폴더째 옮겨도 열림)

### 말풍선
- 한 칸에 여러 말풍선, 드래그 이동·크기 조절
- 모양: 원형/사각, 구름/폭발, 플래시, 집중선, 속도선, 테두리 없음
- 말풍선 꼬리 — 세 점(시작·중간·끝)으로 곡선 조절, 안으로 깎기, 여러 개 추가
- 겹친 말풍선의 외곽선 자동 병합
- 글자색·배경색·아웃라인색

### 저장·내보내기
- 프로젝트 저장/불러오기(`.kfjson`)
- 작업 자동 저장 및 다음 실행 시 복원
- 자동저장을 실행 파일과 같은 폴더에 보관
- 페이지를 이미지 파일로 내보내기
